## 2. Суть проекта

**Telegram-бот для бронирования Scorpion Platinum.**

Цель: чтобы клиенты могли забронировать комнату заранее, не звоня и не переписываясь. Чисто забронировать и прийти.

**Обычный Scorpion НЕ трогаем** — бот только для Platinum.

## 3. Топ-20 идей (все реализованы)

### Ядро — бронирование
1. **Выбор даты и времени** — календарь с доступными слотами
2. **Выбор длительности** — 1ч / 2ч / 3ч / ... с автоматическим расчётом цены
3. **Количество гостей** — проверка вместимости комнаты
4. **Предоплата через бота** — структура готова (ЮKassa / СБП), конфигурируемо
5. **Напоминание за 2 часа** — автоматическое напоминание

### Удобство для клиента
6. **Допы при бронировании** — кальян (вкус), напитки, еда — заказать заранее
7. **Мои бронирования** — история + активные, отмена/перенос
8. **Программа лояльности** — каждый 5-й визит скидка, баллы за визиты
9. **Реферальная система** — персональный код, deep link
10. **"Как в прошлый раз"** — повтор прошлой брони одной кнопкой

### Коммуникация
11. **Уведомления об акциях** — автоматические промо по будням вечером
12. **Отзыв после визита** — рейтинг + комментарий через 30 мин после окончания
13. **Связь с админом** — пересылка сообщений менеджеру

### Админ-панель (/admin)
14. **Просмотр броней** — сегодня/неделя с деталями
15. **Блокировка дат** — закрыть день + причина
16. **Статистика** — загрузка, выручка, средний чек, популярные часы/дни, рейтинг
17. **Чёрный список** — авто (3+ неявки) + ручной

### Фишки для роста
18. **День рождения** — уведомление за 7 дней, скидка 20% ±3 дня
19. **Групповые бронирования** — через вместимость + менеджер
20. **Витрина** — прайс, игры PS5, меню кальянов, еда, геолокация

## 4. Техническая реализация

### Стек
- **Python 3.12** + **aiogram 3.15.0** (async Telegram bot framework)
- **aiosqlite 0.20.0** — SQLite (лёгкая БД, без внешних зависимостей)
- **APScheduler 3.10.4** — шедулер для напоминаний/промо

- **Bot ID**: 8298303089

### Структура проекта
```
scorpion_platinum_bot/
├── main.py              # Точка входа — запуск бота
├── config.py            # ВСЯ конфигурация (цены, часы, адрес, меню)
├── database.py          # SQLite — все CRUD операции
├── keyboards.py         # Все клавиатуры (inline + reply)
├── states.py            # FSM-состояния для диалогов
├── scheduler.py         # 4 фоновые задачи (напоминания, фидбек, ДР, промо)
├── requirements.txt     # Зависимости
├── handlers/
│   ├── __init__.py      # Регистрация всех роутеров
│   ├── start.py         # /start, регистрация, главное меню
│   ├── booking.py       # Полный флоу бронирования
│   ├── my_bookings.py   # Просмотр/отмена/перенос броней
│   ├── loyalty.py       # Лояльность + рефералы
│   ├── showcase.py      # Витрина (прайс, игры, еда, локация)
│   ├── admin.py         # Полная админ-панель
│   └── feedback.py      # Отзывы + связь с менеджером
```

### База данных (SQLite)
Таблицы:
- **users** — telegram_id, имя, телефон, ДР, реф.код, баллы, визиты, неявки, чёрный список, админ
- **bookings** — дата, время, длительность, гости, цена, скидка, статус, напоминание
- **booking_extras** — допы к брони (кальян, еда, напитки)
- **reviews** — рейтинг + комментарий
- **blocked_dates** — заблокированные даты + причина

### Шедулер (4 задачи)
1. **Каждые 5 мин** — проверка напоминаний (за 2ч до визита)
2. **Каждые 10 мин** — запрос отзыва (30 мин после завершения)
3. **Ежедневно 10:00** — промо на ДР (за 7 дней)
4. **Пн–Чт 18:00** — промо свободных вечерних слотов

### Как запустить
```bash
cd scorpion_platinum_bot
py -3 -m pip install -r requirements.txt
py -3 main.py
```

### Первый админ
Первый кто напишет `/admin` боту — автоматически станет администратором.
Или добавь ID вручную в `config.py` → `ADMIN_IDS`.
